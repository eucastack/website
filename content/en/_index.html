What is EucaStack?

EucaStack extends the open source Eucalyptus and Midonet projects to provide an on-premise cloud you can run in your datacenter. Having an on-premise private cloud with AWS-compatible APIs creates a hybrid cloud experience for organizations running workloads in public AWS regions. {{% /blocks/lead %}} {{< blocks/section type="features" color="secondary" >}} {{% blocks/feature icon="fas fa-server fa-2x" title="Private Cloud" height="auto" color="blue" %}} An enterprise-grade private cloud that manages KVM hypervisors in scale. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-id-card fa-2x" title="Multi-Tenancy" height="auto" color="blue" %}} A complete multi-tenancy RBAC-enabled IAM (identity and access management) system. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-code fa-2x" title="Unified APIs" height="auto" color="blue" %}} One set of feature-rich APIs to access a wide range of AWS resources from on-premise to public cloud. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="300" >}} {{% blocks/feature icon="fab fa-discord" title="Join us on Discord!" url="https://discord.gg/A9UPtck4" url_text="Invite link" %}} Chat with other project developers. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/eucastack" %}} We do a [Pull Request](https://github.com/eucastack/eucalyptus/pulls) contributions workflow on **GitHub**.
